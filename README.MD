# Seeker
## _Linux Digital Forensics toolkit written in Python and C_

## Features

- Automated data collection
- Data visualization
- IOC Detection using VirusTotal's API
- Web history analysis

## Built with

- [Streamlit](https://streamlit.io)
- [VirusTotal](http://virustotal.com)
